1- چون از 
api
 سایت باما استفاده شد بعد از چندین بار ریکوئست زدن این سایت درخواست هارو از آیپی مشخص بن میکرد به همین علت از پراکسی استفاده شد و در کد ساختار استفاده از پراکسی مشخص است و داوری کد نیز با استفاده از همین پراکسی ها قابل انجام است.

2- برای آنکه بتوان از پراکسی ها بتوان استفاده کرد باید از اینترنت همراه اول یا مخابرات استفاده شود یا اینکه ابتدا در سیستم خود باید یک پراکسی مانند 
v2ray
استفاده کرد و سپس کد را ران کرد.

3- چون تمرینات قبلی با استفاده از کتابخانه 
requests 
بود در این تمرین سعی شد به صورت  
async 
درخواست ها ارسال شود.

4- برای تغییرات در اطلاعات لاگین دیتابیس لطفا در فایل
database_class.py
اطلاعات متغیرهای کلاسی را مطابق خواسته خود تغییر دهید.

5-  به علت زیاد بودن حجم دادگان و شاید مشکل در دریافت داده های سایت باما 
فایل با پسوند .sql
موجود در پروژه فایل جدول و تمامی دادگان استخراج شده این سایت می باشد.
برای استفاده از این فایل حتما از use 
و اسم دیتابیس مورد نظر خود استفاده کنید.

6- برای قسمت ماشین لرنینگ برای مشخص بودن فرآیند انجام کار از نوت بوک استفاده گردید
